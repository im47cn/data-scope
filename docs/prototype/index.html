关于本原型

这是数据管理与查询系统的交互原型，展示了系统的主要功能和用户界面。本原型包含以下页面：

  • 仪表盘：系统首页，显示关键指标和最近活动
  • 数据源管理：配置和管理数据库连接，查看元数据同步状态
  • 数据浏览：浏览和预览数据表的结构和内容
  • 查询构建器：通过可视化界面构建SQL查询，无需编写代码
  • 自然语言查询：使用自然语言描述数据需求，系统自动转换为SQL
  • 低代码集成：将查询结果无缝集成到低代码平台，快速构建应用
  • 查询历史：查看和管理历史查询记录，重用常用查询
  • 系统设置：配置系统参数、外观和用户偏好

本原型仅用于展示界面设计和交互流程，不包含实际功能。点击各个卡片可以查看对应的界面原型。

多数据源集成

支持MySQL和DB2数据库系统的无缝集成，自动提取和管理元数据，无需手动配置表结构。

智能查询能力

结合SQL查询构建器和自然语言处理，让不同技术水平的用户都能高效查询和分析数据。

低代码集成

通过JSON协议与低代码平台无缝集成，加速应用开发，提供多种数据呈现形式。
